# Effective Learning Techniques

This is a playground note for agent testing. It includes **bold emphasis**, *italic emphasis*, `inline code`, and links back to [[project-ideas]].

## Spaced Repetition

Review material at increasing intervals to move information into long-term memory.

## Active Recall

Test yourself on material rather than passively re-reading.

![Recall diagram](effective-learning-techniques.assets/recall-diagram.jpg)

## The Feynman Technique

1. Choose a concept
2. Explain it simply
3. Identify gaps
4. Simplify and refine
